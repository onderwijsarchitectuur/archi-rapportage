## Archi voorbeeld voor Markdown rapportage.

Deze repo bevat een Archi model met een voorbeeld van een rapportage.

Een script geleend van de Gemma maakt er Markdown van. Het resultaat is in principe te verwerken in Confluence. 

Het werkt met een zogenaamde driving view. Dit is [hier](https://www.tamebluelion.co.uk/archimate-documentation) toegelicht.
